# 📊 Social Media Engagement Analytics (Python Project)

## 📌 Overview

This project analyzes social media engagement data using Python to uncover patterns in user behavior, content performance, and platform trends. The dataset includes metrics such as likes, comments, shares, impressions, watch time, and engagement rate.

The analysis focuses on data cleaning, exploratory data analysis (EDA), statistical analysis, and visualization to generate actionable insights for improving content strategy.

---

## 🎯 Objectives

* Analyze user engagement across different content types
* Identify high-performing posts and categories
* Understand user behavior and engagement patterns
* Evaluate impact of demographics and device usage
* Generate insights to improve social media performance

---

## 🛠️ Tools & Technologies

* Python
* Pandas & NumPy
* Matplotlib & Seaborn
* Plotly (Interactive Visualizations)
* Jupyter Notebook / Google Colab

---

## 📂 Dataset

* `social_media_engagement_5000.csv`
* Contains 5000 records with 19 features including:

  * User demographics (age, gender, country)
  * Engagement metrics (likes, comments, shares)
  * Content details (post type, category, hashtags)
  * Performance indicators (engagement rate, impressions)

---

## 🧹 Data Cleaning & Preparation

* Handled missing values using median/mode techniques
* Removed duplicate records
* Standardized categorical values (gender, sentiment)
* Converted date column to datetime format
* Extracted hashtag count feature
* Corrected inconsistent or unrealistic values

---

## 📊 Exploratory Data Analysis

* Dataset structure and summary statistics
* Distribution of categorical variables
* Correlation analysis between engagement metrics
* Grouped analysis using `groupby()`
* Created new features such as engagement score

---

## 📈 Statistical Analysis

* Mean, median, mode
* Standard deviation and variance
* Percentile distribution
* Analysis of engagement metrics and followers

## 📊 Data Visualization

### Matplotlib

* Scatter plot (Likes vs Impressions)
* Line chart (Engagement trends over time)
* Bar chart (Posts by category)
* Pie chart (Gender distribution)
* Histogram (Age distribution)
* Box plot (Engagement rate)

### Seaborn

* Count plot (Post types)
* Bar plot (Average likes by category)
* Violin plot (Followers vs Sentiment)
* Pair plot (Numeric features)
* Heatmap (Correlation matrix)
* Swarm plot (Engagement vs Device type)

### Plotly

* Interactive charts for deeper exploration

---

## 📊 Key Insights

### 🔥 Content Performance

* Certain post types generate significantly higher engagement
* Specific content categories consistently outperform others
* Engagement varies across countries

### 👥 User Trends

* Age has a noticeable impact on engagement levels
* Verified users tend to receive higher engagement

### 📱 Behavioral Insights

* Engagement varies by time of posting
* Device type influences watch time and interaction

### 💬 Sentiment Analysis

* Positive sentiment posts perform better
* Neutral and negative posts show lower engagement patterns
